Call Analytics

Full Visibility Into Every Conversation

Automatic cost tracking, transcripts and token usage — persisted at call end for billing, compliance and QA.

Full Call Cost Breakdown

Every call is automatically costed — STT per minute, TTS per million characters and LLM input + output tokens — giving precise per-call billing visibility.

Conversation Transcripts

A complete Guest / Agent labeled transcript is generated and saved for every call — enabling QA review, compliance logging and guest interaction analysis.

Duration & Token Tracking

Tracks total call duration, LLM input / output tokens, STT audio duration and TTS character count per session — all persisted to the database at call end.

Configurable Cost Parameters

All cost rates (STT per minute, TTS per million chars, LLM input/output per million tokens) are set via environment variables — update pricing instantly without code changes.

PostgreSQL Session Persistence

Every session ends with an automatic write to the call_conversation table — storing transcript, costs, duration, and token usage with zero manual intervention.

About

The Last Hotel Phone System
You'll Ever Need

Forget hold music. Forget undertrained staff fumbling through menus at 2 AM. The GIT Voice Agent is a blazing-fast, production-hardened AI concierge that picks up every single call — instantly, intelligently, and with the warmth of a five-star front desk. This isn't a chatbot. It's a hospitality revolution.

🎤

Superhuman Guest Lookup

Retrieves any guest's name, check-in date, and contact details in milliseconds — by room number or name. No more awkward silences, no more shuffling papers.

🍽️

Room Service, Reimagined

Guests browse the full menu and place orders conversationally — no app, no downloads, no friction. Just speak. The kitchen gets notified. It's that seamless.

🏨

Concierge at Scale

Pool hours, spa bookings, local hidden gems, housekeeping — handled instantly, 24/7, without a single transfer or minute on hold. One agent. Infinite capacity.

⚡

Voice-First, Always

Engineered from the ground up for natural phone conversations — not repurposed chatbot tech. Deep Gram-enhanced STT, OpenAI LLM, and ultra-low latency TTS combine for eerily human dialogue.

🏗️

Enterprise-Grade Architecture

Modular, clean, and built to last. Centralized config, Docker-ready deployment, SIP trunk support, and PostgreSQL persistence — production-ready out of the box.

🔌

Infinitely Extensible

Swap LLMs, swap STT providers, add new tools — without touching the core. Built on Python 3.10+ with a plugin-friendly design that grows with your property.
